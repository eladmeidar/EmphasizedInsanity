<a href="http://savethedevelopers.com/">savethedevelopers.com</a> used to be a great script that pops up when the client's browser was IE6, warning him about the mischief ways of that browser and urging the user to either update to IE7 or go the happy road on Firefox.<br />
Than, something weird happend.<br />
The website stopped working and now redirects to Microsoft's IE download page... yak.<br />
<br />
The "<a href="http://www.dansimard.com/not_tested_in_ie/">not tested in IE initiative</a>" tries to re-incarnate that thought. So if you are unsure that your website will be displayed correctly on IE6, use one of these icons link to the project's home page.<br />
