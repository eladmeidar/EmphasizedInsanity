<cite>Source: <a href="http://http://woork.blogspot.com/2009/01/10-beautiful-web-ui-libraries.html">Woork!</a></cite><br />
<div class="bigInt">Are you looking for any interesting and beautiful Web User Interface libraries for your next web site or Rich Internet Application ? Take a look at this list with ten WUI I suggest you to try in order to simplify your work and develope faster beautiful and usable user interfaces.</div><br />
<br />
<img src="http://lh5.ggpht.com/_TqPdHmAEwTM/SXj0UDyosCI/AAAAAAAADjs/RoRNxLAp9eo/f1.png" style="border: 1px solid rgb(222, 222, 222); float: right; margin-left: 14px; padding: 6px;" /><strong>1. IT Mill Toolkit</strong><br />
<a href="http://www.itmill.com/itmill-toolkit/">IT Mill Toolkit</a> is an open-source framework, providing widgets and tools for the development of Rich Internet Applications (RIAs). Deliver web applications without worrying about incompatibilities of web browsers, DOM or JavaScript by using standard Java tools.<br />
<br />
<img src="http://lh4.ggpht.com/_TqPdHmAEwTM/SXj-zDVbYwI/AAAAAAAADkc/grpWNZNjQsY/f10.png" style="border: 1px solid rgb(222, 222, 222); float: right; margin-left: 14px; padding: 6px;" /><strong>2. LivePipe UI</strong><br />
<a href="http://livepipe.net/">LivePipe UI</a> is a suite of high quality widgets and controls for web 2.0 applications built using the Prototype JavaScript Framework. Each control is well tested, highly extensible, fully documented and degrades gracefully for non JavaScript enabled browsers where possible<br />
<br />
<img src="http://lh6.ggpht.com/_TqPdHmAEwTM/SXj7UaBwHbI/AAAAAAAADkU/qxanLvoHy8E/f6.png" style="border: 1px solid rgb(222, 222, 222); float: right; margin-left: 14px; padding: 6px;" /><strong>3. Iwebkit iPhone/iPod touch framework</strong><br />
<a href="http://www.iwebkit.net/">Iwebkit</a> is the revolutionnairy kit used to create high quality iPhone and iPod touch websites in a few minutes. In the first 4 months of it's existance the pack has greatly evolved from a basic idea to a project that has reached worldwide fame!<br />
<br />
<img src="http://lh4.ggpht.com/_TqPdHmAEwTM/SXj3eEpXuuI/AAAAAAAADj8/e7bfrShW7y8/f3.png" style="float: right; margin-left: 14px;" /><strong>4. Jitsu</strong><br />
<a href="http://www.jitsu.org/">Jitsu</a> contains an integrated set of tools to enable developers to build and deploy sophisticated user interfaces for web applications. These include an Xml markup language, page compiler, data binding engine, JavaScript runtime, control library, runtime inspector, animation engine, cross-platform library, Ajax, and back button support.<br />
<br />
<img src="http://lh4.ggpht.com/_TqPdHmAEwTM/SXj45ImHEUI/AAAAAAAADkE/8GBdq86XzfU/f4.png" style="border: 1px solid rgb(222, 222, 222); float: right; margin-left: 14px; padding: 6px;" /><strong>5. MochaUI </strong><br />
<a href="http://mochaui.com/">MochaUI</a> is a web applications user interface library built on the Mootools JavaScript framework.Uses: web applications, web desktops, web sites, widgets, standalone windows and modal dialogs.<br />
<br />
<strong>6. Echo Web Framework</strong><br />
<a href="http://echo.nextapp.com/site/">Echo</a> is an open-source framework for developing rich web applications. From the developer's perspective, Echo behaves as a user interface toolkit--like Swing or Eclipse SWT. AJAX technology is employed to deliver a user experience to web clients that approaches that of desktop-based applications.<br />
<br />
<img src="http://lh5.ggpht.com/_TqPdHmAEwTM/SXj2GWo_xBI/AAAAAAAADj0/zbDTiTnPPrI/f2.png" style="border: 1px solid rgb(222, 222, 222); float: right; margin-left: 14px; padding: 6px;" /><strong>7. The Yahoo! User Interface Library (YUI)</strong><br />
<a href="http://developer.yahoo.com/yui/">The YUI Library</a> is a set of utilities and controls, written in JavaScript, for building richly interactive web applications using techniques such as DOM scripting, DHTML and AJAX. YUI is available under a BSD license and is free for all uses.<br />
<br />
<strong>8. Sigma Ajax UI builder</strong><br />
Written in javascript and PHP, <a href="http://sourceforge.net/projects/ajaxuibuilder/">SigmaVisual</a> is web based visual AJAX UI builder for professional developers. Developers save their time in building up prototype as well as real web applications. TreeBar, TreeGrid, Layout, Menu are supported.<br />
<br />
<img src="http://lh5.ggpht.com/_TqPdHmAEwTM/SXj5sJkwGcI/AAAAAAAADkM/r88gAryG5q8/f5.png" style="float: right; margin-left: 14px;" /><strong>9. WUI Web UI Framework</strong><br />
<a href="http://wui.sourceforge.net/">WUI</a> (Web User Interface) is an MVC framework for writing web UIs in a single language: Java. Write web apps with components, widgets &amp; events, not JSPs. Runs in any servlet 2.3 container. Similar to Model 2 / Struts, only better. Apache-style license.<br />
<br />
<strong>10. Butterfly Web UI</strong><br />
<a href="http://butterfly.jenkov.com/webui/index.html">Butterfly Web UI</a> is a component oriented web framework for Java, like Wicket or Tapestry. The main advantage compared to these frameworks is that Butterfly Web UI integrates naturally with Butterfly DI Container, giving you a state-of-the-art dependency injection container to help you structure and decouple the internal components of your web applications.
